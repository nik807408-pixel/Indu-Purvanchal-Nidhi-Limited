# Indu Purvanchal Nidhi Limited — Finance Management PWA

Setup steps:
1. Supabase project banao (region: Mumbai ap-south-1)
2. SQL Editor me `supabase-setup.sql` poora run karo
3. Authentication → Sign In/Providers → Email → "Confirm email" OFF
4. `app.js` me line 5-6 par apna SUPABASE_URL aur anon KEY paste karo
5. GitHub repo me push → Settings → Pages → deploy from main branch
6. Client app me signup kare → SQL se admin banao (setup.sql STEP 6)
